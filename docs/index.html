PersistentImages_Chemistry Documentation:¶

The PersistentImages_Chemistry repository contains tools to create persistence images (PI) to encode geometric data of chemical structures for use in machine learning applications. An overview of PI’s and a basic tutorial of generating a persistence image is included in this tutorial.

The source code for this repository can be found in the original repository.

You can download the original paper at this link: Original Chemistry PI Paper

Installation¶

Install this code by entering the following commands:

git clone https://github.com/maroulaslab/PersistentImages_Chemistry.git
