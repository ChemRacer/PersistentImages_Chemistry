PersistentImages_Chemistry Documentation:¶

The PersistentImages_Chemistry repository contains tools to create persistence images (PI) to encode geometric data of chemical structures for use in machine learning applications. An overview of PI’s and a basic tutorial of generating a persistent diagram and a persistence image is included in this tutorial.

The source code for this repository can be found in the original repository.
